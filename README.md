# Spaarnelanden Containers (Home Assistant)

Home Assistant integration to track filling degree of Spaarnelanden containers.

## Features
- UI-based setup wizard
- Enter one or more container registration numbers manually
- Comma-separated input (e.g. 150484,20126,20137)
- 1-click install via HACS
- No YAML required

## Installation
1. Add repository https://github.com/tgriek/ha-spaarnelanden-containers as a HACS Integration
2. Install and restart Home Assistant
3. Settings → Devices & Services → Add Integration → Spaarnelanden Containers
